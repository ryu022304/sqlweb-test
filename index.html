How to check IndexedDB status

  1. Open developper tools (Win: Ctr+Shift+i, Mac: Option+Command+i)
  2. Go to Application tab(Chrome) or Storage tab(Firefox)
  3. Check items of IndexedDB

Available SQL example

  • select * from TABLENAME
  • select * from TABLENAME where COL = 'VAL'
  • insert into TABLENAME value(COL1: VAL1, COL2: VAL2, COL3: VAL3)
  • count * from TABLENAME
