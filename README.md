sqlweb-test
====
This is a test program for SqlWeb.

## Description
You can see that you can use SQL to operate IndexedDB by SqlWeb.

## Requirement
* npm
* webpack

## Installation
```
$ clone https://github.com/ryu022304/sqlweb-test
$ cd sqlweb-test
$ npm i
$ webpack
```

## Usage
1. Open index.html in a browser
2. Input SQL query in form
3. Click EXEC button
